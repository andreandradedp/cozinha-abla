# Cozinha ABLA

Este é o repositório para o sistema de controle de presenças e alimentação escolar.

## Funcionalidades
- Cadastro de escolas e salas.
- Registro diário de presenças.
- Controle de dietas específicas.
- Painel para gestora da cozinha.
